## 试用方法

1. 克隆 git 仓库到本地
2. 将 sql 文件指令导入 MySQL（使用其它数据库请载入对应的驱动jar）

端口号 | 用户名 |  密码  | 数据库名
------|--------|--------|---------
 3306 |  root  | 123456 | final
 
3. 在 Eclipse 中导入 JavaWeb 项目
4. 启动 Tomcat 服务器进行调试
